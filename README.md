## Lab 1
**upd 2025.11.15**: 已上传 Lab 1 的 `bits.c`（即主代码文件）, 后面将其复制进虚拟机中的 `bits.c` 即可。复制完成后，需在终端运行一些命令。   

首先要确保在存放文件的路径里。这里我直接将文件存放在主目录下。  

执行如下代码
```bash
$ ./dlc bits.c
```

这段代码用来进行规则检查。由于题目限制了运算符的类型和个数，因此通过该命令，可以检查运算符的使用是否合法。

随后可以进行如下命令，进行编译：
```bash
$ make btest
```
编译通过后，可以进行正确性检查：
```bash
$ ./btest
```
如果检查没问题，就可以直接上传了：
```bash
$ ./driver.pl  -u  学号 –p 密码
```
然后就OK了~
